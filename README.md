## **兼容性**
* 仅在 PHP 7.2 以上进行过测试。

## **插件列表**

* 百度百科
* 点歌（网易音乐、QQ音乐、咪咕音乐、哔哩哔哩音乐）
* 群打卡 or 群签到
* 一言（[Hitokoto](https://hitokoto.cn/)）
* 毒鸡汤
* 葫芦侠三楼图片
* 帮我选
* 地震通报

## **注意事项**
* 计划任务功能依赖于go-cqhttp的心跳事件，请勿关闭！
